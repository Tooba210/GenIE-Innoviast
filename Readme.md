# ◆ GenIE - Enterprise AI Content Studio

## 🎯 Overview
**GenIE** (Generative Intelligent Engine) is an enterprise-grade AI content generation studio that creates high-quality structured content for 6 different use cases using Groq's Llama 3.3 70B model.

## ✨ Features

### 📝 6 Content Templates
- Blog Post
- LinkedIn Post  
- Email
- Ad Copy
- Product Description
- Caption

### 🎛️ Smart Controls
- **8 Tone Options:** Professional, Casual, Persuasive, Emotional, Humorous, Formal, Inspirational, Witty
- **3 Length Options:** Short, Medium, Long
- **4 Output Formats:** Paragraphs, Bullet Points, Numbered List, JSON Format
- **Target Audience:** Custom audience targeting

### 🎨 Professional UI
- Clean, minimal corporate design
- Enterprise-grade interface
- Responsive layout
- Professional toolbar with copy/download options

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| Python | Backend Logic |
| Streamlit | UI Framework |
| Groq API | AI Content Generation (Llama 3.3 70B) |
| dotenv | Environment Variables |
